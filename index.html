Foodie Customer Review System

My role in this project was to solely implement the backend webservices, databases and the web application using java, Spring, Hibernate, JSP and jQuery. The webservices was used by an android application loaded on a tablet device to get data from customers.

This application was designed to allow hotel and restaurant owners to receive customer feedback data during billing. Analyzing Safety Metrics of Major US Towns/Cities Using Semantic Web

Based on the data received from various government agencies regarding demography, education, crimes and federal expenditures, applied semantic web engineering to generate safety index of all major cities of US.

My tasks involved gathering historical data from sources like FBI, UCR, data.world, NCES, Census Bureau, etc. Then I designed a semantic data model with relationships between all the data columns using apache Jena framework to make the data more usable. After this the data was converted into RDF instances using this data model.

These RDF instances were then served on an apache Fuseki server. This server was used to query the RDF data efficiently and provide a good estimate about safety index of cities along with providing insights on the socio-economic conditions which might affect the crime rates.

Created an AngularJS based web app which allowed users to enter names of the US cities. Based on this, queried the Fuseki endpoint to display all the relevant information to the user.
